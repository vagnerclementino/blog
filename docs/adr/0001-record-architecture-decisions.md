# 1. Registrar decisões de arquitetura

Date: 2026-06-01

## Status

Accepted

## Context

Precisamos registrar as decisões de arquitetura tomadas neste projeto.

## Decision

Utilizaremos Architecture Decision Records (ADRs), conforme [descrito por Michael Nygard](http://thinkrelevance.com/blog/2011/11/15/documenting-architecture-decisions).

## Consequences

Veja o artigo de Michael Nygard, referenciado acima. Para um conjunto leve de ferramentas de ADR, veja [adr-tools](https://github.com/npryce/adr-tools) de Nat Pryce.
